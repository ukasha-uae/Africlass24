# PWA Assets Needed

To complete the PWA setup, please add the following files:

## Icons
- `public/icons/icon-192x192.png` (192x192 pixels)
- `public/icons/icon-512x512.png` (512x512 pixels)

## Screenshots (Optional but recommended)
- `public/screenshots/mobile.png` (Mobile view screenshot)
- `public/screenshots/home.png` (Desktop/Wide view screenshot)

You can generate these icons using tools like [RealFaviconGenerator](https://realfavicongenerator.net/).
